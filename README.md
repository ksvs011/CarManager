# CarManager
### 주요기능
- 차량을 주차, 출차, 주차 공간 현황 확인 가능
- dataGridView에서 입력한 차량정보 확인 가능
- ListBox에서 로그 기록 확인 가능
- 지정된 경로에 파일이 없어도 자동 txt파일 생성
